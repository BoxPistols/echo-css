l-row

floatによるグリッドのグループを作成する。

補足：floatによるグリッドの特徴

  • .l-row と .l-col が直接の親子関係ではない場合でもグリッドが成立する
  • セルの開始位置をずらすことができる
  • Internet Explorer 9以前のバージョンでも利用できる
  • セル内のコンポーネントの高さを揃えることができない
  • セルの上下揃えを指定できない
